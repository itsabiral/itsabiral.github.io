created a resume maker using typescript, vanillajs, jquery, html and bootstrap, made an api using typescript to accept resume information as json and generate a resume in the backend.<br>
update: added openai api as option to generate resume texts on demand<br>
added new user selective elements to the page using document object model.<br>
also, stored and configured user's resume in a server to make it downloadable to the users.
