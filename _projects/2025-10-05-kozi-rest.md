developed a sleep environment analysis system using raspberry pi 4 with a ccs811 air quality sensor and ultrasonic movement detection.<br>
used python, node.js, express, mongodb, and socket.io to connect from the iot layer to the server and display data in a real-time dashboard.<br>
a real-time data processing with a node.js backend, socket.io for live updates, and mongodb for data storage.<br>
ai-powered morning reports using gemini for analysis and elevenlabs tts for audio feedback via a bluetooth speaker.<br>
created a web dashboard with chart.js visualizations, auth0 authentication, and real-time sensor data streaming.<br>
